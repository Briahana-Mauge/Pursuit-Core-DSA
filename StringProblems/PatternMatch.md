# Pattern Match

| Difficulty | Tags |
| --- | --- | --- | 
| Easy | [Arrays, Strings, Dictionaries]


Write a func that takes in a pattern: "aba" and words and returns whether the words appear in the order of the pattern.

## Sample

| Input | Output |
| --- | --- |
|"ece", "apple banana apple" | true |
|"fff", "can bag dog" | false |
