## Problem Description

Given a square, two dimensional array of Ints, find the row or column that has the greatest sum.  If more than one row or column are tied, return all of them.

## Sample

| Input | Output |
| --- | --- |
| 1 2 9 <br> 8 2 3 <br> 4 5 6 | Column 2: 18 |
