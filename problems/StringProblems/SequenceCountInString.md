# Sequence Count in String

| Difficulty | Tags |
| --- | --- | --- | 
| Medium | [Arrays, Integers]

## Problem Description

Given a string, return a comma separated string that contains the counts of each character in a row.

## Sample

| Input | Output |
| --- | --- |
|"abccdacc" | "1a, 1b, 2c, 1d, 1a, 2c" |
